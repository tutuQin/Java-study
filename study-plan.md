# 🎯 你的“达标标准”（重点）

你不需要会写所有代码，但必须做到：

- ✅ 看懂 Spring Boot 项目结构
    
- ✅ 能改接口逻辑（哪怕是AI生成的）
    
- ✅ 能用 MySQL 查数据、定位问题
    
- ✅ 能看懂并调整 MyBatis 的SQL
    
- ✅ 能把项目用 Docker 跑起来
