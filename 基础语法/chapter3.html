第三章 运算符

3.1 运算符概览

Java 提供了丰富的运算符,可分为以下七大类:

Java 运算符
├── 算术运算符 ── 二元: + - * / %        一元: ++ --
├── 赋值运算符 ── = += -= *= /= %=  &= |= ^= <<= >>= >>>=
├── 关系运算符 ── == != > >= < <=  instanceof
├── 逻辑运算符 ── 逻辑: & | ^ !          短路: && ||
├── 三元运算符 ── 条件 ? 值1 : 值2
├── 位运算符   ── & | ^ ~ << >> >>>
└── 字符串连接 ── +

按操作数个数划分:

  • 一元运算符:1 个操作数,如 ++a、!flag、-x
  • 二元运算符:2 个操作数,如 a + b、a == b
  • 三元运算符:3 个操作数,只有 ? : 一个

3.2 算术运算符

① 二元算术运算符

运算符含义示例结果
+加5 + 27
-减5 - 23
*乘5 * 210
/除5 / 22(整数除法,截断小数)
%取余(模)5 % 21
 
System.out.println(10 / 3);     // 3   (整数除法,只保留整数部分)
System.out.println(10.0 / 3);   // 3.3333333333333335
System.out.println(10 % 3);     // 1   (取余)
System.out.println(-10 % 3);    // -1  (符号取决于被除数)
System.out.println(10.5 % 3);   // 1.5 (浮点数也可取余)
⚠️ 注意:整数除以 0 会抛出 ArithmeticException;浮点数除以 0 不会异常,而是得到 Infinity 或 NaN。 
int x = 10 / 0;        // ❌ ArithmeticException: / by zero
double y = 10.0 / 0;   // ✅ Infinity
double z = 0.0 / 0;    // ✅ NaN

② 一元算术运算符:自增 ++ 与自减 --

分为前缀和后缀两种形式,作用是把变量值 ±1。

形式含义取值时机
++a / --a前缀先自增/自减,再使用值
a++ / a--后缀先使用值,再自增/自减
 
int a = 5;
int b = ++a;     // a 先变 6,再赋值给 b → a=6, b=6

int c = 5;
int d = c++;     // c 先赋值给 d,再变 6 → c=6, d=5

// 经典面试题
int i = 1;
int r = i++ + ++i + i++;   // 1 + 3 + 3 = 7, 最终 i=4
💡 建议:不要在一个表达式中多次使用同一变量的自增/自减,代码难读且易错。把它们拆成单独语句。

3.3 赋值运算符

赋值运算符把右边的值赋给左边的变量。复合赋值运算符 (+=, -= 等) 是 "运算 + 赋值" 的简写。

运算符等价于示例
=—a = 10
+=a = a + ba += 5
-=a = a - ba -= 5
*=a = a * ba *= 5
/=a = a / ba /= 5
%=a = a % ba %= 5
 
int a = 10;
a += 5;      // a = 15
a -= 3;      // a = 12
a *= 2;      // a = 24
a /= 4;      // a = 6
a %= 4;      // a = 2
📌 隐式类型转换:复合赋值运算符自带强制转换,不会因类型不匹配报错。 
byte b = 10;
// b = b + 1;  ❌ 错误!b+1 自动提升为 int
b += 1;            // ✅ 编译通过,等价于 b = (byte)(b + 1)

3.4 关系运算符(比较运算符)

关系运算符用于比较两个值,结果一定是 boolean 类型 (true / false)。

运算符含义示例结果
==等于5 == 5true
!=不等于5 != 3true
>大于5 > 3true
>=大于等于5 >= 5true
<小于5 < 3false
<=小于等于3 <= 5true
instanceof判断对象是否属于某类型"abc" instanceof Stringtrue
 
int a = 10, b = 20;
System.out.println(a == b);    // false
System.out.println(a < b);     // true

// instanceof 用法
Object obj = "Hello";
if (obj instanceof String) {
    String s = (String) obj;
}
// JDK 16+ 模式匹配,自动绑定变量
if (obj instanceof String s) {
    System.out.println(s.length());
}
⚠️ 重点区分 == 和 .equals():
  • 基本类型用 == 比较值
  • 引用类型用 == 比较内存地址,用 .equals() 比较内容
  • 比较字符串内容必须用 .equals()
String s1 = new String("abc");
String s2 = new String("abc");
System.out.println(s1 == s2);          // false (不同对象)
System.out.println(s1.equals(s2));     // true  (内容相同)

3.5 逻辑运算符

逻辑运算符用于连接布尔表达式,结果仍为 boolean。分为普通逻辑和短路逻辑。

运算符名称含义示例
&逻辑与两边都为 true 才为 true,两边都要计算a & b
|逻辑或有一边为 true 即为 true,两边都要计算a | b
^逻辑异或两边不同为 true,相同为 falsea ^ b
!逻辑非取反!a
&&短路与左边为 false 时,右边不再计算a && b
||短路或左边为 true 时,右边不再计算a || b

真值表

aba&ba|ba^b!a
truetruetruetruefalsefalse
truefalsefalsetruetruefalse
falsetruefalsetruetruetrue
falsefalsefalsefalsefalsetrue

短路特性(重要)

int a = 5;
int b = 10;

// 短路与: a>0 为 true,继续判断 b>0
if (a > 0 && b > 0) { ... }

// 经典应用: 避免空指针
String s = null;
if (s != null && s.length() > 0) {   // ✅ 安全
    System.out.println(s);
}
if (s != null & s.length() > 0) {    // ❌ NullPointerException
    System.out.println(s);
}

// 短路或: 第一个为 true,后面不执行 i++
int i = 0;
boolean ok = (10 > 5) || (i++ > 0);
System.out.println(i);   // 输出 0,i++ 没有执行
💡 实战建议:绝大多数场景应使用短路运算符 && 和 ||,效率更高且能避免空指针。只有当确实需要执行右边表达式(如带副作用的方法)时才使用 & 和 |。

3.6 三元运算符(条件运算符)

Java 中唯一的三元运算符,是 if-else 的简洁写法。

// 语法:  布尔表达式 ? 值1 : 值2
// 若布尔表达式为 true,返回 值1;否则返回 值2

int a = 10, b = 20;
int max = (a > b) ? a : b;       // max = 20

String result = (score >= 60) ? "及格" : "不及格";

// 可以嵌套(但不要嵌太深)
String level = score >= 90 ? "优秀"
              : score >= 60 ? "及格"
              : "不及格";
📌 注意:
  • "值1" 和 "值2" 必须是兼容类型,否则编译错误
  • 结果类型为两个值的共同父类型(自动提升)
  • 三元表达式必须有返回值,不能用作单独语句
// ❌ 错误用法
// (a > b) ? max=a : max=b;   编译错误,不能作为语句

// 类型提升示例
Object r = (true) ? 1 : "hello";   // 类型为 Object

3.7 位运算符

位运算符直接操作二进制位,常用于底层开发、位标志、性能优化等场景。

运算符名称说明示例(a=12, b=10)
&按位与两位都为 1 才为 112 & 10 = 8
|按位或有一位为 1 即为 112 | 10 = 14
^按位异或两位不同为 112 ^ 10 = 6
~按位取反0 变 1,1 变 0~12 = -13
<<左移高位丢弃,低位补 0,相当于 ×2ⁿ12 << 2 = 48
>>右移(算术)低位丢弃,高位补符号位,相当于 ÷2ⁿ12 >> 2 = 3
>>>无符号右移低位丢弃,高位补 0-1 >>> 28 = 15
 
// 12 = 0000 1100      10 = 0000 1010
System.out.println(12 & 10);   // 0000 1000 = 8
System.out.println(12 | 10);   // 0000 1110 = 14
System.out.println(12 ^ 10);   // 0000 0110 = 6

// 实用技巧
int a = 5, b = 7;
a = a ^ b;  b = a ^ b;  a = a ^ b;   // 不借助第三方变量交换两个数

int n = 10;
boolean isEven = (n & 1) == 0;    // 判断奇偶,比 % 更快
int doubled = n << 1;              // 乘 2,比 *2 更快

3.8 字符串连接符 +

+ 在 Java 中有两种作用:数字相加 和 字符串拼接。当任一操作数为 String 时,+ 表示拼接,其他操作数自动转为字符串。

System.out.println(1 + 2);              // 3 (加法)
System.out.println("abc" + 123);        // "abc123" (拼接)
System.out.println("结果=" + 1 + 2);    // "结果=12" 从左到右
System.out.println(1 + 2 + "=" + 3);    // "3=3"
System.out.println(1 + "" + 2);          // "12" (常用 +"" 转字符串)
⚠️ 性能提示:循环中频繁使用 + 拼接字符串会创建大量临时对象,应使用 StringBuilder: 
StringBuilder sb = new StringBuilder();
for (int i = 0; i < 1000; i++) {
    sb.append(i).append(",");
}
String result = sb.toString();

3.9 运算符优先级

运算符按优先级从高到低排列(只需记住常用规则,复杂表达式建议加括号)。

优先级运算符结合性
1(最高)() [] .左→右
2! ~ ++ -- 一元 + -右→左
3* / %左→右
4二元 + -左→右
5<< >> >>>左→右
6< <= > >= instanceof左→右
7== !=左→右
8&(按位与)左→右
9^(按位异或)左→右
10|(按位或)左→右
11&&左→右
12||左→右
13? :(三元)右→左
14(最低)= += -= *= /= 等赋值右→左
 
// 示例:体会优先级
int r1 = 2 + 3 * 4;          // 14 (先乘后加)
int r2 = (2 + 3) * 4;        // 20 (括号优先)
boolean r3 = 5 > 3 && 2 < 1; // 关系优先于 &&
int r4 = 1 + 2 << 3;         // (1+2)<<3 = 24,+ 优先于 <<
💡 终极建议:不要靠记忆优先级表,用括号让意图明确,既不会出错,代码也更易读。
