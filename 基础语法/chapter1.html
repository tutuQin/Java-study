1.1 Java 数据类型概览

Java 是强类型语言,任何变量必须先声明类型再使用。Java 数据类型分为两大类:

  • 基本数据类型(Primitive Type):共 8 种,直接存储数值本身,存放在栈内存。
  • 引用数据类型(Reference Type):如类、接口、数组、字符串等,变量存储的是对象在堆内存中的地址。
Java 数据类型
├── 基本类型 (8 种)
│   ├── 整数型: byte、short、int、long
│   ├── 浮点型: float、double
│   ├── 字符型: char
│   └── 布尔型: boolean
└── 引用类型
    ├── 类 (Class) 如 String
    ├── 接口 (Interface)
    └── 数组 (Array)
💡 提示:基本类型不是对象,但 Java 为每一种基本类型都提供了对应的包装类(Wrapper Class)以支持面向对象操作。

Java 共有 8 种基本数据类型,它们的字节数、取值范围都是固定的,不随操作系统而变化(这是 Java 跨平台的重要保证)。

1.2.1 整数类型(byte / short / int / long)

Java 中所有整数类型都是有符号的(最高位为符号位)。默认整数字面量类型为 int,如需 long 必须在数字末尾加 L(推荐大写,小写 l 易与数字 1 混淆)。

public class IntegerDemo {
    public static void main(String[] args) {
        byte  b = 100;            // -128 ~ 127
        short s = 30000;
        int   i = 2147483647;     // int 最大值
        long  l = 9999999999L;    // 超过 int 范围必须加 L

        // 不同进制写法(JDK 7+)
        int dec = 100;            // 十进制
        int bin = 0b1100100;       // 二进制 0b 前缀
        int oct = 0144;            // 八进制 0 前缀
        int hex = 0x64;            // 十六进制 0x 前缀

        // 数字下划线分隔符(JDK 7+),便于阅读
        long big = 1_000_000_000L;
    }
}
⚠️ 注意:整数溢出不会抛出异常,而是发生回绕。例如 Integer.MAX_VALUE + 1 会得到 Integer.MIN_VALUE。

1.2.2 浮点类型(float / double)

Java 浮点数遵循 IEEE 754 标准。默认浮点字面量为 double,声明 float 必须在末尾加 F 或 f。

float  f1 = 3.14F;          // 必须加 F
double d1 = 3.14;           // 默认就是 double
double d2 = 3.14D;          // 加 D 可选

// 科学计数法
double e1 = 1.5e3;          // 1500.0
double e2 = 2.5E-2;         // 0.025

// 特殊值
double inf = 1.0 / 0.0;    // Infinity (正无穷)
double nan = 0.0 / 0.0;    // NaN (Not a Number)
⚠️ 精度陷阱:浮点数不能精确表示所有十进制小数。例如 0.1 + 0.2 != 0.3。需要精确计算(如金额)时,务必使用 java.math.BigDecimal。
System.out.println(0.1 + 0.2);   // 输出: 0.30000000000000004

// 正确做法:使用 BigDecimal,且必须用字符串构造
BigDecimal a = new BigDecimal("0.1");
BigDecimal b = new BigDecimal("0.2");
System.out.println(a.add(b));   // 输出: 0.3

1.2.3 字符类型(char)

char 占 2 字节,使用 Unicode 编码(UTF-16),可表示中文、英文、符号等。字符字面量用单引号包裹,且只能包含一个字符。

char c1 = 'A';             // 字符
char c2 = '中';             // 中文字符
char c3 = 65;              // 用整数赋值,自动转为 'A'
char c4 = 'A';         // Unicode 转义,等价于 'A'
char c5 = '\n';            // 转义字符:换行

// char 可参与运算,会自动转为 int
int sum = 'A' + 1;        // 结果是 int 66
char next = (char)('A' + 1); // 强制转换得到 'B'

1.2.4 布尔类型(boolean)

布尔类型只有两个值:true 和 false。Java 中布尔与整数不能相互转换(这与 C/C++ 不同)。

boolean flag = true;
boolean isReady = 10 > 5;   // 表达式结果

// ❌ 错误写法:不能用整数代替布尔
// if (1) { ... }   编译错误

if (flag) {
    System.out.println("执行了");
}

1.3 引用数据类型(Reference Types)

除 8 种基本类型外,其他都是引用类型。引用变量存储的是对象的内存地址(引用),对象本身存放在堆中。

  • 类(Class):如 String、Object、自定义类等
  • 接口(Interface):如 List、Runnable
  • 数组(Array):如 int[]、String[]
String name = "Tom";          // 字符串
int[]  arr  = {1, 2, 3};       // 数组
Object obj  = new Object();     // 对象
List<Integer> list = new ArrayList<>(); // 集合

String s = null;             // 引用类型可以为 null
💡 区别:基本类型 == 比较的是值,引用类型 == 比较的是地址。比较内容应使用 .equals() 方法。

1.4 数据类型默认值

类的成员变量(字段)如果没有显式赋值,JVM 会自动赋予默认值;但局部变量没有默认值,使用前必须显式初始化。

类型默认值
byte, short, int, long0
float0.0f
double0.0d
char' ' (空字符)
booleanfalse
引用类型null
 
public class DefaultDemo {
    int field;                // 成员变量,默认 0
    String name;             // 成员变量,默认 null

    public void test() {
        int local;            // 局部变量,无默认值
        // System.out.println(local); ❌ 编译错误: variable local might not have been initialized
        local = 10;
        System.out.println(local);
    }
}

1.5 数据类型转换

① 自动类型转换(隐式转换)

小范围类型 → 大范围类型,自动完成,不会丢失精度。

byte → short → int → long → float → double
              ↑
            char →
    
int i = 100;
long l = i;          // int → long,自动转换
double d = l;        // long → double

char c = 'A';
int code = c;        // char → int,自动得到 65

② 强制类型转换(显式转换)

大范围类型 → 小范围类型,必须显式强转,可能丢失精度或溢出。

double d = 3.99;
int i = (int) d;     // 结果是 3 (截断,非四舍五入)

int big = 200;
byte b = (byte) big;  // byte 最大 127,结果溢出为 -56

long l = 10000000000L;
int n = (int) l;     // 高位被截断

③ 表达式自动提升

byte a = 1, b = 2;
// byte c = a + b;  ❌ 错误!a+b 自动提升为 int
int  c = a + b;       // ✅ 正确
byte d = (byte)(a + b); // ✅ 或强转
📌 规则:任何 byte、short、char 参与运算时都会先提升为 int。混合运算时,结果类型为参与运算的最高类型。

④ 字符串与基本类型转换

// 基本类型 → 字符串
String s1 = String.valueOf(123);
String s2 = Integer.toString(123);
String s3 = 123 + "";             // 简便写法

// 字符串 → 基本类型(使用包装类的 parseXxx 方法)
int    n = Integer.parseInt("123");
double d = Double.parseDouble("3.14");
boolean b = Boolean.parseBoolean("true");

1.6 包装类(Wrapper Class)

Java 为每种基本类型提供了对应的包装类,使基本类型也能以"对象"的形式参与运算(如放入集合、使用泛型等)。

基本类型包装类基本类型包装类

intIntegercharCharacter
longLongbooleanBoolean

自动装箱与拆箱(Autoboxing / Unboxing)

JDK 5 引入了基本类型与包装类之间的自动转换。

// 自动装箱: int → Integer
Integer i = 100;          // 等价于 Integer.valueOf(100)

// 自动拆箱: Integer → int
int n = i;                  // 等价于 i.intValue()

List<Integer> list = new ArrayList<>();
list.add(1);                // 自动装箱
int first = list.get(0);  // 自动拆箱
⚠️ 注意:Integer 缓存了 -128 ~ 127 的对象,这个范围内 == 比较为 true,超出范围则为 false。比较 Integer 值请用 .equals() 或 .intValue()。
Integer a = 100, b = 100;
System.out.println(a == b);          // true  (缓存范围内)

Integer c = 200, d = 200;
System.out.println(c == d);          // false (超出缓存)
System.out.println(c.equals(d));     // true  (正确比较)

字面量指代码中直接写出的固定值。
